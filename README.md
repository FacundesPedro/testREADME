### To Run a nodejs http-Server

# On bash
```
    //Importante de setar variaveis de ambiente,se não PORT=8080
    PORT=ENV_PORT_VALUE node index.mjs

```
# On windows
```
    //Importante de setar variaveis de ambiente, se não PORT = 8080
    $env:PORT 8080
    node .\index.mjs //env "PORT" setada anteriormente!!
```
